VPS FINANCIAL SUITE - TOOLBOX MODUL
ANWENDERDOKUMENTATION
====================================

ÜBERBLICK
---------
Das VPS Toolbox Modul ist eine Sammlung praktischer Hilfswerkzeuge für SAP Business One. Das Hauptfeature ist die "Alle Zeilen löschen"-Funktion, die über Rechtsklick-Kontextmenüs in Matrix-Formularen zugänglich ist. Diese Funktion ermöglicht es, alle Zeilen in einer Matrix mit einem Klick zu entfernen, anstatt jede Zeile einzeln löschen zu müssen.

GRUNDLAGEN UND INFOS ZUM MODUL
------------------------------
Das Toolbox Modul erweitert automatisch alle SAP Business One Formulare um die "Alle Zeilen löschen"-Funktion. Wenn Sie in einem Formular mit einer Matrix (Tabelle) arbeiten, erscheint beim Rechtsklick ein zusätzlicher Menüeintrag, mit dem Sie alle Zeilen der Matrix auf einmal entfernen können.

Das Modul arbeitet vollautomatisch und erfordert keine Konfiguration. Es registriert sich für alle Formulare in SAP Business One und reagiert auf Rechtsklick-Ereignisse.

HAUPTFUNKTIONEN
---------------
AUTOMATISCHE ERKENNUNG:
- Erkennt automatisch Matrix-Objekte in SAP B1 Formularen
- Fügt die "Alle Zeilen löschen"-Option zum Kontextmenü hinzu
- Löscht alle sichtbaren Zeilen einer Matrix mit einem Klick
- Arbeitet mit allen Standard-SAP-B1-Formularen
- Erkennt automatisch, welche Matrix gerade aktiv/sichtbar ist
- Prüft, ob die Matrix bearbeitbar ist
- Wählt die optimale SAP-Standard-Löschfunktion aus

SICHERHEITSMECHANISMEN:
- Verwendet nur Standard-SAP-Löschfunktionen
- Keine direkten Datenmanipulationen
- Respektiert Formular-Berechtigungen
- Funktioniert nur bei bearbeitbaren Matrizen

FUNKTIONSWEISE IM DETAIL
------------------------
Das Modul verwendet eine intelligente Technik zur Zeilen-Löschung:

ZEILEN-ERKENNUNG:
- Das System ermittelt automatisch die Anzahl sichtbarer Zeilen
- Identifiziert die erste bearbeitbare Spalte
- Prüft, welche Standard-SAP-Löschfunktionen verfügbar sind

LÖSCH-ALGORITHMUS:
1. Das System klickt auf die erste bearbeitbare Zelle
2. Aktiviert die entsprechende SAP-Standard-Löschfunktion
3. Wiederholt den Vorgang für alle verbleibenden Zeilen
4. Nutzt entweder "Zeile löschen" oder "Entfernen" je nach Verfügbarkeit

AUTOMATISCHE BEREINIGUNG:
- Event-Handler werden nach jedem Rechtsklick wieder entfernt
- Keine permanenten Menü-Änderungen
- Temporäre Kontextmenü-Erweiterungen

ALLGEMEINE ANWENDUNG
--------------------
Die Funktion zum Löschen aller Zeilen ist einfach zu verwenden und steht in allen Matrix-basierten Formularen zur Verfügung. Sie aktiviert sich automatisch, sobald Sie ein entsprechendes Formular öffnen.

GENAUE SCHRITT FÜR SCHRITT ANLEITUNG FÜR ALLE FUNKTIONEN
---------------------------------------------------------
SO FUNKTIONIERT ES:
1. Öffnen Sie ein beliebiges SAP Business One Formular mit einer Matrix (z. B. Ausgangsrechnung, Bestellung, etc.)
2. Rechtsklick auf die Matrix (Tabelle mit den Zeilen)
3. Wählen Sie "Alle Zeilen löschen" aus dem Kontextmenü
4. Alle Zeilen in der Matrix werden automatisch entfernt
5. Eine Bestätigungsmeldung erscheint in der Statusleiste

VORAUSSETZUNGEN:
- Das Formular muss eine bearbeitbare Matrix enthalten
- Sie müssen über die entsprechenden Bearbeitungsrechte verfügen
- Das Formular darf nicht im Nur-Lese-Modus geöffnet sein

ZEITERSPARNIS:

FEHLERBEHANDLUNG BEI PROBLEMEN
-------------------------------
Die häufigsten Fragen und Lösungen:

Problem: Kontextmenü erscheint nicht
Lösung: Stellen Sie sicher, dass Sie in einem Formular mit einer Matrix sind und diese bearbeitbar ist

Problem: Funktion löscht nicht alle Zeilen
Lösung: Das ist normal — das System verwendet SAP-Standard-Funktionen, die bei bestimmten Zeilen nicht greifen können

Problem: Fehlermeldung beim Löschen
Lösung: Prüfen Sie, ob Sie Bearbeitungsrechte für das Formular haben

Problem: Matrix reagiert nicht / Matrix friert ein
Lösung: Das System arbeitet mit Formular-Sperrung (Freeze/Unfreeze) — warten Sie kurz, bis der Vorgang abgeschlossen ist

STATUSMELDUNGEN:
- "Alle Zeilen löschen : Abgeschlossen" — Erfolgreiche Ausführung
- Bei Fehlern erscheint eine entsprechende Fehlermeldung in der Statusleiste

ZUSAMMENFASSUNG
---------------
Das VPS Toolbox Modul ist ein praktisches Hilfswerkzeug, das die tägliche Arbeit mit SAP Business One erheblich erleichtert. Die "Alle Zeilen löschen"-Funktion spart Zeit und reduziert Aufwand bei der Bearbeitung von Matrix-basierten Formularen.
