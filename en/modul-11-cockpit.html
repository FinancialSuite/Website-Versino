Cockpit

Overview

The Versino Cockpit is the central dashboard of the Financial Suite and offers a comprehensive working environment for managing, analyzing and entering postings. It runs as a separate window in parallel to SAP Business One and acts as a starting point and control center for many financial processes.

Access: The Cockpit is reachable via Versino Financial Suite > Financial Cockpit. It automatically takes over your SAP B1 session and user permissions.

Important Distinction: "Posting Stack" vs. "Posting Batch"

The Cockpit has two similar-sounding but very different areas:

  • Account Sheets > Posting Stack: ONLY display & analysis of existing postings (dynamic primary note).
  • Posting Batch tab: Tool to CREATE new journal entries (BookingStackView).

Main Features

Home Page (Launchpad)

The home page is your personal dashboard with configurable tiles for quick access to the most important areas of SAP Business One.

[IMAGE: Home page of the Cockpit with all 7 tile areas and labeled categories]

Account Sheets (Analyze Existing Postings)

Under the tab "Account Sheets" you will find a powerful view of journal entries with reference to accounts/customers/vendors/primary note (DATEV). This is a pure analysis and read view that allows you to quickly filter and sort existing postings and to display details by double-clicking. No new postings can be created here.

[IMAGE: Account Sheets > Posting Stack with dynamic primary note — example of filter and sort functions]

Posting Batch (Quickly Enter New Journal Entries)

Under the tab "Posting Batch" you will find a separate tool that is specifically designed for the fast creation of new journal entries. It offers an entry interface with intelligent automation:

  • Automatic account-type detection (expense, revenue, etc.)
  • Automatic tax assignment based on the account master data
  • Automatic gross/net calculation
  • Enter-key handling for fast entry
  • Auto-locking of fields for filled-in data

Mandatory fields: Account, Posting Date, Document Type, Offset Account, Posting Text, Amount.
Optional fields: Reference 1–3, Project plus dynamic cost-center fields based on SAP B1 dimensions.

[IMAGE: Posting Batch tool with filled-in mandatory fields and grid before posting creation]
[IMAGE: Side-by-side comparison: left Account Sheets Posting Stack (display), right Posting Batch tool (creation)]

Documents (Creating Parked Documents)

Under the tab "Documents" you will find a separate tool that is specifically designed for creating documents from parked documents.

  • After marking the rows, you can add or remove the parked documents via the Continue to button.
  • After document creation you receive a status message with all successful or failed documents.
  • Subsequently you are automatically redirected to the Documents tab and all created documents are displayed.

Login (Multi-Company Function)

Under the tab "Login" you can log into another company — either via the current window or via a new window.

Predefined Queries

Under the tab "Queries", the following predefined analyses are available:

  • Tax-exempt BPs (BPFreed) — Overview of all tax-exempt business partners
  • Tax-liable BPs (BPMandatory) — Overview of all tax-liable BPs
  • Dunning Lock Customers (DunLock) — Customers with active dunning lock
  • Payment Lock Vendors (PayLock) — Vendors with active payment lock
  • A/P Invoice Control (APInvoiceControle) — Control and analysis of A/P invoices
  • Evaluation by Account (EvaluationByAccount) — Detailed account analyses with country breakdown
  • Accounts without Default Tax (AccountVsTax) — Identification of tax inconsistencies
[IMAGE: Queries tab with the listing of all predefined queries]

Reports in the Cockpit

Under the tab "Reports", the Cockpit-internal reports are available:

  • BWA — Business analysis with HANA and SQL Server support
  • TB — Classic trial balance evaluation
  • OPK — Open items customers with due-date analyses

In addition, in the tab "Master Data" comprehensive business partner overviews are available with direct editing capability via the context menu (right-click).

Integration with Other VPS Modules

VPS Main Module & Configurations

Via the home page tiles in the FS area, you reach all configurations of the Financial Suite directly — DATEV Export/Import, exchange rates, Pro-Rata settings, JE text configuration, XsAdmin.

VPS Reporting

The Cockpit reports BWA, TB and OPK access the central reporting components and automatically use the optimized versions for SQL Server and HANA respectively.

SAP B1 Multi-Company

The Cockpit allows multiple parallel sessions in different companies. Tip: In the Cockpit, you are not subject to license limitations — you can start more than 2 sessions with the same user simultaneously.

Application

Starting the Cockpit

  1. Navigate to Versino Financial Suite > Financial Cockpit.
  2. The Cockpit opens as a separate Windows window.
  3. The home page with the tile areas is displayed automatically.

Using the Posting Batch Tool for Journal Entries

  1. Switch to the Posting Batch tab.
  2. Fill in the mandatory fields: Account, Posting Date, Document Type, Offset Account, Posting Text, Amount.
  3. Optional: Add reference fields and project.
  4. Press Enter or click "Get Data" to add to the grid.
  5. Repeat for further posting rows.
  6. Click "Functions > Create Journal Entry". The posting is automatically opened in SAP Business One.

Using the Dynamic Primary Note

  1. Switch to the Account Sheets tab.
  2. Choose "Posting Stack" for the dynamic primary note view.
  3. All journal entries are displayed in a clearly arranged table.
  4. Use column sorting, filters and double-click on rows for detail views.

Multi-Company Login

  1. Switch to the Login tab.
  2. Choose the desired company from the dropdown list.
  3. Enter the username and password.
  4. Optional: Activate "Log in in a new window".
  5. Click "Login" for the connection.

Tips and Troubleshooting

Tips for Daily Work

  • Summation tip: Mark several rows with numerical values in any table. The sum of these values is automatically displayed when hovering on the column header.
  • Login tip: When re-logging in, you remain logged in to the current B1 session. In the Cockpit, you are not subject to license limitations — you can start more than 2 sessions with the user.
  • Use Posting Stack areas correctly: Use the dynamic primary note (Account Sheets) for posting analyses, the Posting Batch tool for creating new journal entries.
  • Personalization: Configure the home page according to your work habits and use the tile management for an individual interface.
  • Performance: Close unused tabs for better performance.

Common Problems and Solutions

Problem: Confusion between the Posting Stack areas.

Solution: Note the difference — Account Sheets show existing postings, Posting Batch tab creates new postings.

Problem: Posting Batch tool entry does not work.

Solution: Check all mandatory fields (Account, Posting Date, Document Type, Offset Account, Posting Text, Amount) and validate the account master data.

Problem: Dynamic primary note shows no data.

Solution: Check the database connection, validate the user permissions for journal entries and check the filter settings.

Problem: Cockpit does not start.

Solution: Restart SAP Business One as administrator and check the VPS Financial Suite installation.
