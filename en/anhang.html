Modules at a Glance (Quick Reference)

Quick access to all 11 modules of the Financial Suite with their SAP B1 menu paths and main purposes:

Module SAP B1 Menu Path Main Purpose
Financial Suite (Main Module) Versino Financial Suite > Configuration > Financial Suite Wizard / Settings Central configuration, wizard, netting, client/consultant data
DATEV Export Versino Financial Suite > DATEV Export Export of postings, master data and payment terms to DATEV format
DATEV Import Versino Financial Suite > Import/Export > Import Import of DATEV CSV batches (pre-entry or direct posting)
Reporting Package Versino Financial Suite > Financial Suite Reports Standard and mass reports (BWA, TB, balance confirmation, payment advice and more)
Period Accrual/Deferral (Pro-Rata) Versino Financial Suite > Configuration > Pro-Rata Settings Day-precise accrual/deferral postings for contracts, licenses, insurances
Exchange Rates Import Versino Financial Suite > Import Exchange Rates / Configuration > Exchange Rate Settings Automatic rate import from ECB and NBP, hourly or manual
Posting Text Configurator Versino Financial Suite > Configuration > JE Text Configuration Automatic, uniform posting texts for all document types
Bank Statement Import Versino Financial Suite > Bank Statement Import Import of electronic bank statements (MT940/CAMT) with RegEx engine
Fast Entry A/P Invoice Purchasing > Fast Entry A/P Invoice Create recurring vendor invoices from templates
Toolbox Right-click in any matrix table "Delete all rows" for matrix tables in SAP B1
Cockpit Versino Financial Suite > Financial Cockpit Central dashboard, posting batch, multi-company login

General Information

Licensing

All modules are licensed as FullAddon and require corresponding user licenses.

Compatibility

  • Supports both SQL Server and SAP HANA.
  • Compatible with On-Premise and Cloud installations.
  • Works with SAP Business One Client and Web Access.
